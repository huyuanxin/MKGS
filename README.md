# 工程简介

## 快速开始
 * 启动相关依赖（两个图形数据库和solr）
 * 启动项目
 * 启动前端
## 模块介绍

* knowledge-admin 后台数据管理模块
* knowledge-base-dep 依赖管理模块
* knowledge-common 公共模块
* knowledge-disease-diagnosis 疾病诊断模块
* knowledge-question-and-answer 智能问答模块
* knowledge-starter 项目启动模块

## 技术选型

### 前端

* Vue-Element-Admin

### 后端

* Spring Boot 2.3.0+
* MySQL 5.7+
* Neo4j 4.2.1+
* Solr 7.3.0+

### 基本功能

* 知识问答
* 疾病分析
